# 准备工具
- Nvm
	- 简介：用来管理Nodejs版本的工具
	- 安装：
		- Windows：GitHub项目上安装指定版本
	- 使用
```
安装nodejs最新版：nvm install latest
安装指定版本：nvm install vX.Y.Z
指定默认版本：nvm alias default vX.Y.Z
切换版本：nvm use vA.B.C
```
